# OCR PNG to Excel - 快速使用指南

⚡ **30 秒快速上手** | 适合首次使用的用户

## 🎯 一句话介绍

将图片中的表格自动识别并转换为可编辑的 Excel 文件。

## 📖 三步完成识别

### 第 1 步：上传图片
- 拖拽图片到上传区域，或点击"选择文件"
- 支持格式：PNG、JPG、JPEG 等

### 第 2 步：开始识别
- 点击"开始识别"按钮
- 等待 10-30 秒（根据图片大小）

### 第 3 步：下载 Excel
- 识别完成后，可在线编辑（双击单元格）
- 点击"保存修改"（如有编辑）
- 点击"下载 Excel"

## ⚠️ 注意事项

1. **图片要求**
   - ✅ 清晰、光线充足、无倾斜
   - ✅ 文件小于 10MB
   - ❌ 避免模糊、过暗的图片

2. **编辑技巧**
   - 双击单元格进行编辑
   - Enter 保存，Esc 取消
   - 编辑后记得点"保存修改"

3. **性能建议**
   - 单次上传一张图片
   - 处理复杂表格时耐心等待
   - 定期清理旧文件释放空间

## 🆘 遇到问题？

### 服务无法启动
```bash
# 检查端口是否被占用
netstat -tuln | grep -E '3000|8000'

# 停止冲突进程后重启
./scripts/stop.sh
./scripts/start.sh
```

### 识别失败
- 检查图片质量（清晰度、光线）
- 重新拍摄或扫描图片
- 联系技术支持
